## General Information

- Multilinear Regression Model is developed for Bike sharing Company

- Bike Rental Dataset with variables - 'instant', 'dteday', 'season', 'yr', 'mnth', 'holiday', 'weekday',
  'workingday', 'weathersit', 'temp', 'atemp', 'hum', 'windspeed',
